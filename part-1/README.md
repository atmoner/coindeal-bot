## Requirements

 - Valid coindeal account
 - Nodejs and npm
 - Some free time


## Installation

```sh
git clone https://github.com/atmoner/coindeal-bot.git
cd coindeal-bot/part-1/
npm i
```
Run it
```
node myBot.js
```
